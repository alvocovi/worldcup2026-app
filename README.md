# FIFA World Cup 2026 App ⚽

A React app showing all 104 FIFA World Cup 2026 matches with per-country broadcast information.

## Features

- 📅 Full schedule — all 72 confirmed group stage matches + complete knockout bracket
- 🌍 95 countries covered with TV channels and streaming platforms
- 📺 Per-match channel info (BBC vs ITV for UK, FOX vs FS1 for US, Globo vs SBT for Brazil)
- 🔍 Searchable country dropdown with region filter
- ⏰ All kick-off times auto-converted to your device's local timezone
- 🏆 Filter by phase (Group Stage, R32, R16, QF, SF, Final)
- 🔎 Search by team name to highlight their matches

## Getting Started

Drop `worldcup2026.jsx` into any React project as the default export, or open it in a sandbox:

- [StackBlitz](https://stackblitz.com) — paste as `App.jsx`
- [CodeSandbox](https://codesandbox.io) — create a React project and replace `App.js`

No external dependencies required beyond React itself.
